## 1. 專案概述

### 1.1 背景
現有 `travel_app` 提供基礎的翻譯、換匯、路線規劃功能，但缺乏完整的行程規劃能力、離線支援、預算管理、以及智慧推薦系統。本專案目標是將現有應用升級為「自助旅遊神器」，提供一站式的旅遊規劃與執行體驗。

### 1.2 目標
- 建立完整的行程規劃引擎（AI 驅動）
- 支援離線 PWA 模式
- 整合預算追蹤與花費分析
- 強化智慧推薦系統（景點、美食、住宿）
- 即時語音翻譯與對話輔助
- 地圖整合與即時導航
- 天氣預報整合
- 社群分享與行程匯出

## 3. 功能規格

### 3.1 核心功能 (Must Have)

#### 3.1.1 行程規劃引擎
- **輸入**: 目的地、天數、預算、興趣標籤
- **輸出**: 每日行程表（含景點、美食、交通、時間安排）
- **AI 驅動**: 使用 Qwen3.6 生成個性化行程
- **格式**: JSON 結構化輸出，支援匯出 PDF/CSV

#### 3.1.2 離線 PWA 模式
- **Service Worker**: 快取 API 回應與靜態資源
- **離線行程查看**: 已規劃行程可離線存取
- **離線地圖**: 快取區域地圖瓦片
- **離線翻譯**: 快取常用詞彙與短語

#### 3.1.3 預算追蹤器
- **預算設定**: 總預算、分類預算（住宿、餐飲、交通、購物）
- **花費記錄**: 手動/自動記錄每筆花費
- **即時匯率**: 自動轉換當地貨幣
- **圖表分析**: 圓餅圖、折線圖顯示花費分佈
- **超支警告**: 預算剩餘 20% 時提醒

#### 3.1.4 智慧推薦系統
- **景點推薦**: 基於興趣、評分、距離推薦
- **美食推薦**: 基於菜系、預算、評分推薦
- **住宿推薦**: 基於位置、價格、評分推薦
- **隱藏景點**: AI 推薦非主流但高品質景點

### 3.2 進階功能 (Should Have)

#### 3.2.1 即時語音翻譯
- **雙向對話**: 支援即時語音輸入與輸出
- **多語言**: 支援 10+ 語言
- **語境理解**: AI 理解對話語境，提供更自然翻譯
- **文化提示**: 提供當地禮儀與文化注意事項

#### 3.2.2 地圖整合與導航
- **即時定位**: GPS 定位使用者位置
- **路線規劃**: 步行、騎車、開車、大眾運輸
- **POI 搜尋**: 搜尋附近景點、餐廳、加油站
- **導航模式**: 轉彎提示、距離提醒

#### 3.2.3 天氣整合
- **即時天氣**: 顯示當前天氣狀況
- **預報**: 7 天天氣預報
- **穿衣建議**: 根據天氣推薦穿著
- **活動建議**: 根據天氣推薦適合的活動

### 3.3 擴充功能 (Could Have)

#### 3.3.1 社群分享
- **行程分享**: 生成分享連結或 QR Code
- **社群登入**: Google/Facebook 登入
- **評論系統**: 使用者可評論景點與餐廳
- **行程模板**: 分享行程模板供他人使用

#### 3.3.2 多使用者支援
- **行程協作**: 多人共同編輯行程
- **角色權限**: 管理者、編輯者、查看者
- **版本控制**: 行程版本管理

#### 3.3.3 進階分析
- **花費預測**: AI 預測總花費
- **時間優化**: 優化行程時間安排
- **碳足跡計算**: 計算旅行碳足跡

## 4. 非功能需求

### 4.1 效能需求
- **API 回應時間**: < 2 秒（一般請求）
- **LLM 回應時間**: < 10 秒（簡單問題）
- **頁面載入時間**: < 3 秒（首次）
- **離線模式**: 核心功能 100% 離線可用

### 4.2 可用性需求
- **響應式設計**: 支援手機、平板、桌面
- **無障礙**: WCAG 2.1 AA 等級
- **多語言 UI**: 繁體中文、英文、日文

### 4.3 安全性需求
- **CORS 設定**: 限制來源
- **輸入驗證**: 防止 SQL Injection、XSS
- **API 限流**: 防止濫用
- **資料加密**: 敏感資料加密儲存

## 7. 技術查證報告

### 7.1 Ollama & Qwen3.6
- **狀態**: 2026 年最新穩定版
- **模型**: Qwen3.6 26B 參數，支援多語言
- **效能**: 本地執行，延遲 < 2 秒（簡單問題）
- **建議**: 維持現狀，無需升級

### 7.2 FastAPI
- **狀態**: 0.115+ 為最新穩定版
- **特性**: 原生 Async 支援、自動文件生成
- **建議**: 維持現狀

### 7.3 Leaflet.js
- **狀態**: 1.9.4 為最新穩定版
- **替代方案**: MapLibre GL JS（更現代化）
- **建議**: 維持現狀，除非需要 3D 地圖

### 7.4 PWA 技術
- **狀態**: Service Worker API 成熟
- **瀏覽器支援**: Chrome 90+, Safari 14+, Firefox 89+
- **建議**: 採用標準 PWA 實作

### 7.5 天氣 API
- **選項**: Open-Meteo（免費、無需 API Key）
- **替代**: WeatherAPI.com（免費額度有限）
- **建議**: 採用 Open-Meteo
